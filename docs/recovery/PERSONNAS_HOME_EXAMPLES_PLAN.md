# Personnas Home Examples Plan

## But

Preparer la future section d'accueil inspiree de Character.AI, sans copier son interface.

L'objectif est de montrer rapidement a l'utilisateur :
- ce qu'est une persona SocioSim
- comment on peut l'explorer
- pourquoi le projet n'est pas seulement une liste de profils, mais une bibliotheque d'experiences d'entretien

## Principe retenu

La page d'accueil devra afficher seulement quelques personnas exemples, pas toute la bibliotheque.

Ces cartes doivent donner envie d'ouvrir une fiche ou de commencer un entretien, tout en restant pedagogiques.

## Personnas exemples pressentis

### Jade

Role :
- montrer un rapport critique, politique et conflictuel a l'IA
- incarner les tensions entre valeurs, usages contraints et engagement

Utilite pour la home :
- persona fort narrativement
- bon exemple pour montrer que les entretiens peuvent produire des contradictions exploitables

### Oriane

Role :
- montrer un rapport plus nuance, introspectif et universitaire a l'IA
- incarner le perfectionnisme, la procrastination et les arbitrages d'etudiante

Utilite pour la home :
- persona lisible pour les etudiants
- bon contraste avec Jade

### Theo ou un persona futur

Role :
- completer la vitrine avec un profil plus pragmatique, technique ou quotidien
- eviter que la home donne l'impression que tous les personnas sont critiques ou universitaires de la meme maniere

Utilite pour la home :
- diversifier les postures
- montrer que la methode peut accueillir plusieurs profils

## Comportement attendu sur la home

Chaque carte exemple devrait proposer :
- nom du persona
- description courte
- tension principale
- action `Voir la fiche`
- action `Commencer un entretien`

Plus tard, le style pourra etre rapproche de l'inspiration Character.AI :
- cartes vivantes et expressives
- avatars ou initials fortes
- contraste entre profils
- hover ou micro-interaction
- mise en scene plus premium

## Ce que le bloc personnas doit deja fournir

Avant de refaire la home, le bloc personnas doit deja garantir :
- une fiche persona lisible
- une grille d'entretien claire
- un prompt associe
- un historique accessible
- un CTA d'entretien stable

Etat :
- base prete
- integration home a faire dans le bloc accueil
